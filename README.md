# Smart NFC & QR Visiting Card Platform

A modern, colorful, and fully responsive frontend for a SaaS platform where users can create, preview, and manage their smart visiting cards with QR and NFC integration.

## Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
